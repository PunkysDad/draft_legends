# Legends Clash Monorepo

This monorepo contains the full stack for **Legends Clash**, a fantasy sports drafting application.

## Structure

- **`frontend/`** — React Native mobile app (TypeScript)
- **`backend/`** — Spring Boot API (Kotlin, Gradle)

## Getting Started

### Backend

```bash
cd backend
./gradlew bootRun
```

### Frontend

```bash
cd frontend
npm install
npx react-native run-ios   # or run-android
```
